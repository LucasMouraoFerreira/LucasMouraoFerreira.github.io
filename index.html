Fake Fish Store

Projeto de um e-commerce, em que é possível realizar operações CRUD sobre as entidades, quando logado na conta administradora. Como cliente é possível criar uma conta, comprar produtos e acessar ordens de pedido. O site também realiza o consumo da API do Google MAPS para mostrar a localização da loja no mapa, web service dos correios para calcular frete, e realiza o processamento do pagamento pelo gateway Stripe.

Tecnologias utilizadas: C#, Asp.net Core MVC, Entity Framework, SQL Server, Javascript, HTML 5 + Razor, CSS 3 e Bootstrap 4.

Deploy: myWindowsHosting.

O que tem de bão em BH?

REST API que retorna lugares de Belo Horizonte. Com a API é possível utilizar os verbos Http POST, GET, PUT e DELETE sobre as entidades Place e Comment, realizar buscas por lugares com base no texto contido no nome ou descrição, região e avaliação média. Também é possível localizar lugares próximos de uma localização, passando como parâmetros uma distância máxima, a latitude e longitude de um ponto ou um endereço. No caso de receber um endereço, uma chamada para a API Geocoding da Google é realizada.

Para evitar o mau uso da API, limitei o acesso a alguns métodos através da autenticação por JWT. Todos os métodos GET de /places e /comments e POST de /comments/addComment/{place_id} estão liberados para testes.

Tecnologias utilizadas: Java, Spring Boot 2, JWT, Spring Security, PostgreSQL, JPA, Maven, Swagger 2.

Deploy: Heroku.
